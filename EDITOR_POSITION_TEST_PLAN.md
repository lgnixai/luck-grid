# 编辑器位置修复 - 测试计划

## 测试目标

验证编辑器在所有场景下都能精确对齐单元格位置，误差不超过 2px。

## 测试环境

- **浏览器**：Chrome、Firefox、Safari、Edge
- **设备**：桌面端、移动端
- **页面布局**：有 toolbar、无 toolbar

## 自动化测试

### 自动验证脚本

页面提供了"自动验证编辑器位置"按钮，执行以下步骤：

1. 滚动到测试单元格
2. 设置为活动单元格
3. 获取期望位置（通过 getCellBounds）
4. 模拟双击进入编辑
5. 获取实际位置
6. 比较期望与实际，计算误差
7. 显示验证结果

### 预期结果

```
=== 自动验证结果 ===

测试单元格: 列1, 行5
命中检测: col=1, row=5 ✅

期望位置:
  top=240.0, left=228.0
  width=240.0, height=40.0

实际位置:
  top=240.0, left=228.0
  width=240.0, height=40.0

误差(px):
  top: 0.0 ✅
  left: 0.0 ✅
  width: 0.0 ✅
  height: 0.0 ✅

最终结果: ✅ 全部通过
```
